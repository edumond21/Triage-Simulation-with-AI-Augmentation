## Overview

SOC//AI puts you in the seat of a tier-1 security analyst. Each session, a fresh queue of **8 randomised alerts** is generated by Gemini AI — covering phishing, malware, network anomalies, and identity threats. For every alert, a second Gemini instance acts as your AI analyst partner, surfacing indicators, reasoning through the evidence, and asking clarifying questions when the picture is ambiguous.

Your job: decide **True Positive**, **False Positive**, or **Escalate**.

When you're done, a results screen grades your performance from *Needs Training* to *Elite Analyst*.

---

## Features

- **AI-generated alert queues** — Every session is unique. No two runs are the same.
- **Three AI analyst modes** per alert:
  - `VERDICT + REASON` — AI gives a definitive TP/FP call with confidence score
  - `INDICATOR HIGHLIGHT` — AI flags suspicious signals without giving the answer away
  - `GUIDED INQUIRY` — AI expresses uncertainty and asks you a clarifying question
- **Live confidence bar** — Visual representation of the AI's certainty
- **Post-verdict explanations** — Learn *why* each alert was a TP or FP after submitting
- **Session results screen** — Score, accuracy %, and breakdown of TP/FP/missed
- **Zero dependencies** — Single HTML file, no build step, no framework

## How It Works

```
Page load
  └─▶ Gemini generates 8 randomised alerts (JSON array)
        └─▶ Each alert has: type, severity, fields, correct answer, AI mode

Select alert
  └─▶ Second Gemini call analyses the alert in the assigned mode
        └─▶ Returns: verdict, confidence %, reasoning, indicator tags

Submit verdict
  └─▶ Score updated → explanation shown → next alert
        └─▶ After alert 8 → finish screen with grade
```

Alert types included:
- **Phishing Email** — typosquatting, BEC, SPF/DKIM failures
- **Malware / Endpoint** — PowerShell LOLBins, ransomware, C2 beacons
- **Network Anomaly** — DNS tunneling, unusual data transfers, port scans
- **Suspicious Login / IAM** — impossible travel, service account abuse, after-hours access

## Self-Hosting

The app is a single HTML file. To run your own copy:

1. Clone the repo:
   ```bash
   git clone https://github.com/edumond21/Triage-Simulation-with-AI-Augmentation.git
   ```

2. Get a **free** Gemini API key (no credit card required):  
   → [aistudio.google.com](https://aistudio.google.com) → **Get API key** → **Create API key**

3. Open `soc_analyst_ai_simulator_v2.html` in a text editor and replace the key value:
   ```javascript
   const GEMINI_KEY = 'YOUR_KEY_HERE';
   ```

4. Open the file in any browser — no server needed.

## Deploying to GitHub Pages

To host your own live version:

1. Go to your repo on GitHub → **Settings** → **Pages**
2. Under *Source*, select **Deploy from a branch**
3. Choose **main** branch, **/ (root)** folder → click **Save**
4. Your live URL will be:  
   `https://<your-username>.github.io/<repo-name>/soc_analyst_ai_simulator_v2.html`

> **API key note:** For public repos, consider restricting your key to your GitHub Pages domain in [Google AI Studio](https://aistudio.google.com) to prevent unauthorised use. Go to your key → **Add restrictions** → **HTTP referrers** → add `https://yourusername.github.io/*`.
